# Sentiment Analysis Using TextBlob

## Project Overview

This project performs sentiment analysis on Twitter text data using Python and the TextBlob library. The objective is to analyze textual content, determine sentiment polarity, and classify tweets as Positive, Neutral, or Negative.

The project demonstrates a complete text analytics workflow, including data loading, text preprocessing, sentiment scoring, sentiment classification, and visualization of sentiment distribution.

## Technologies Used

* Python
* Pandas
* TextBlob
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The dataset consists of text reviews related to Avengers movies. These reviews were used to perform sentiment analysis and classify opinions as Positive, Neutral, or Negative.

## Workflow

1. Load and inspect the dataset.
2. Clean and preprocess tweet text.
3. Calculate sentiment polarity scores using TextBlob.
4. Classify tweets into Positive, Neutral, and Negative categories.
5. Identify the most positive and most negative tweets.
6. Visualize sentiment distribution using charts and graphs.

## Key Features

* Text preprocessing and cleaning
* Sentiment polarity calculation
* Sentiment classification
* Sentiment distribution analysis
* Data visualization
* Extraction of highly positive and highly negative tweets.

## Limitations and Future Improvements

This project uses the TextBlob library for sentiment analysis, which is suitable for learning Natural Language Processing (NLP) fundamentals and understanding sentiment classification workflows.

However, rule-based sentiment analysis has several limitations:

* Difficulty understanding sarcasm and irony.
* Limited ability to interpret context.
* May misclassify complex or ambiguous statements.
* Performance depends on the quality of the input text.
* Sentiment scores may not always reflect human interpretation.

Future improvements could include:

* Using transformer-based models such as BERT.
* Fine-tuning Large Language Models (LLMs) for sentiment classification.
* Incorporating context-aware sentiment analysis techniques.
* Evaluating model performance using benchmark datasets and metrics.
* Building a web application for real-time sentiment analysis.

This project helped in understanding Natural Language Processing (NLP) fundamentals, text preprocessing techniques, sentiment analysis concepts, and data visualization using Python.
